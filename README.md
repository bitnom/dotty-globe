# ThreeJS Globe

## Quick start

```bash
npm install dotty-globe
```

```html
<div id="globe-container"></div>
```

```javascript  
const container = document.getElementById("globe-container")
createGlobe(container, dataPoints)
```

## Develop

```bash
git checkout https://github.com/blairjordan/dotty-globe
npm install
npm run dev
```

## Build

```bash
npm run build
```
